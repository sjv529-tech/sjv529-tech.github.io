📚 Web Dev Resources

a curated collection of tools, tutorials, and references

ella's resource library

here are the resources that helped me learn web development. i hope they help you too! ✧

📖 documentation & references

  • MDN Web Docsessential the definitive resource for HTML, CSS, and JavaScript documentation. maintained by Mozilla and updated regularly.
  • W3Schools beginner-friendly tutorials and interactive examples. great for quick reference and basic concepts.
  • CSS-Tricks comprehensive CSS guides, flexbox/grid visual references, and modern layout techniques.
  • The Modern JavaScript Tutorial in-depth JavaScript coverage from basics to advanced concepts. free and regularly updated.

🎨 design & colour

  • Coolors palette generator for creating harmonious colour schemes. i used this to choose my portfolio colours!
  • Color Hunt curated colour palettes from the community. great for inspiration when you're stuck.
  • Google Fonts free, open-source web fonts. all fonts are optimised for web performance.
  • Coolors Contrast Checker WCAG compliance checker for colour contrast. essential for accessibility.

🛠️ development tools

  • Visual Studio Codemy editor free, extensible code editor. recommended extensions: Live Server, Prettier, ESLint.
  • GitHubessential version control and collaboration platform. free for public repositories and student accounts.
  • Supabase open-source Firebase alternative
